# PK-R Curriculum

A structured, self-paced curriculum for learning R in the context of 
clinical pharmacology, PK/PKPD analysis, and drug development.

## About

Built to develop practical R skills for:
- Noncompartmental analysis (NCA) with PKNCA
- Clinical pharmacology TFLs (tables, figures, listings)
- Population PK modeling with nlmixr2 and mrgsolve
- Exposure-response and exposure-safety analysis
- PBPK and QSP modeling
- R Shiny app development for interactive PK tools

## Structure

| Phase | Modules | Focus |
|-------|---------|-------|
| 1 | M01–M08 | R + GitHub foundations, EDA |
| 2 | M09–M16 | NCA and TFLs |
| 3 | M17–M26 | Population PK and covariate analysis |
| 4 | M27–M36 | PKPD, E-R, and simulation |
| 5 | M37–M44 | PBPK, QSP, and Shiny apps |

## Tools and packages

R 4.4+, RStudio, tidyverse, PKNCA, nlmixr2, mrgsolve, rxode2,
gtsummary, gt, flextable, ggplot2
